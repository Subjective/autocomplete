We are building a local-first macOS assistant that offers system-wide inline tab/edit completion (similar to Github Copilot/Cursor Tab), but across all input fields, with screenshot-derived scene memory to recover recently viewed context such as visible emails, documents, or web pages.

Before making edits, start by reading `docs/feature-list.md` to understand the current state of progress. If you complete any of the features, check them off.
